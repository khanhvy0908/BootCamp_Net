## ▶ How to Run

- Clone the repository  
- Open the solution file in Visual Studio  
- Build and run the project using **F5** (with debugging) or **Ctrl + F5** (without debugging)

### 💻 Requirements
- .NET 6.0 SDK or later  
- Visual Studio 2022 (recommended)
